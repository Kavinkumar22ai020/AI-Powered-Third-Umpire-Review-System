# 🏏 Cricket Third Umpire Decision Review System

A Python-based web application that simulates the cricket third umpire decision review process using video replay and manual decision controls.

---

##  Project Type

- Software / Application Project
- Video Processing + Web UI
- Manual Decision Review System
- **Not an AI / ML prediction system**

---

##  Features

- Upload or play a cricket video
- Frame-by-frame control (Fast / Slow)
- Manual OUT / NOT OUT decision
- Professional web-based UI
- Simulates real third umpire workflow

---

##  Tech Stack

- **Backend:** Python, Flask
- **Frontend:** HTML, CSS
- **Video Processing:** OpenCV
- **Configuration:** YAML
